辞書化したいテキストを貼ってください（Ctrl+D で終了）。

以下の手順で実行してください:

1. AskUserQuestion ツールは使わず、Bash ツールで直接 `cat` コマンドを使ってユーザーの入力を受け取り、`tools/paste_expression.sh` に渡してください。

2. 実行するコマンド（ユーザーが貼り付けて Ctrl+D で送る）:

```bash
bash tools/paste_expression.sh
```

3. 出力をそのままユーザーに見せてください。

4. 成功時（exit 0）: 「インポート完了」とコミット情報を表示する。
5. 失敗時（exit 非0）: エラーログを表示し、「commit/push は行われていません」と伝える。

---

**対応フォーマット（自動判別）:**

| 形式 | 例 |
|---|---|
| 全角カッコ | `日本語（english）` |
| 半角カッコ | `日本語(english)` |
| ダッシュ | `日本語 - english` |
| タブ区切り | `日本語⇥english` |
| コロン（逆順も吸収） | `english : 日本語` |

**安全装置（自動チェック）:**
- 10件未満 → 停止
- 空キー → 停止
- 重複率 > 50% → 停止
- 新規追加 < 5件 → 停止（exit 22）
- JSON破損 → 停止
- 想定外ファイル変更 → 停止
